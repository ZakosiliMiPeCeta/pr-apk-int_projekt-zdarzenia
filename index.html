W języku HTML połączonym z kodowaniem w JS możemy wywoływać różne funkcje za pomocą Eventów, których kilka przykładowych pokażę poniżej:

Click

Działa to w taki sposób, że w przycisku został wpisany parametr onclick="(nazwa funkcji)", który odpala daną funkcję po kliknięciu.

Double Click

Tutaj sytuacja wygląda analogicznie, tylko zamiast onclick() został użyty dblclick()

MouseOver oraz MouseOut

W tym przypadku, zamiast używać parametru dla przycisku, jest użyty EventListener. Jest to druga metoda uruchamiania funkcji w JS. Nadaje się przyciskowi ID, po czym odwołując się do niego dopina się Event listenera, który następnie w odpowiednim warunku uruchomi funkcję. EventListenera można dodać każdemu elementowi, któremu wcześniej nadano ID

MouseUp i MouseDown

Ten przycisk wygląda bardzo podobnie do przycisku z onclick. Różnica jest w tym, że tutaj funkcja uruchomi się zaraz po tym, jak przycisk zostanie odpuszczony albo wciśnięty, a onclick musi zarejestrować mousedown oraz mouseup by wykonać funkcję.

MouseMove

W przypadku mousemove, EventListener nasłuchuje, czy myszka rusza się po danym elemencie.

Select

Event select sprawdza, czy zawartość danego pola tekstowego jest zanaczona.

Focus i blur

Zdarzenie focus sprawdza, czy użytkownik kliknął na dane pole tekstowe, kierując tam kursor używany do wpisywania tekstu.

blur natomiast jest odwrotnością focusa, gdyż sprawdza on, czy opuszczono pole tekstowe.

Change

change wykrywa wszelkie zmiany w polu tekstowym, które zostały potwierdzone klawiszem Enter, lub po zmianie opuszczono pole tekstowe.

Jak to wszystko oskryptować?

Jak wspomniałem wyżej, można zrobić to na 2 sposoby.

Pierwszym jest wpisanie w input parametr on(akcja). Pzykład poniżej:

                    #HTML
                    <input type=button value="Przykładowy przycisk" onclick=funkcja() />

                    #JS
                    function funkcja()  {
                        //Tutaj wpisz co ma wykonać funkcja
                    }

Jest jeszcze druga metoda, dokładniej użycie EventListenera. Poniżej można zobaczyć, jak to działa:

                    #HTML (UWAGA: NALEŻY NADAĆ ID INPUTOWI!)
                    <input type="button" value="Przykładowy przycisk 2" id="idPrzycisk" />

                    #JS
                    let przycisk = document.getElementById("idPrzycisk");
                    przycisk.addEventListener(click, [nazwa funkcji BEZ ARGUMENTÓW AKTUALNYCH]);

                    function [nazwa funkcji]() {
                        //Co ma ta funkcja wykonać
                    }
